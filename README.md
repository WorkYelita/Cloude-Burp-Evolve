# manager-proyectos
Sistema de gestión de proyectos con Claude

## Qué es este repositorio

Este repositorio es la memoria viva del proyecto HookSuite. Aquí se documenta absolutamente todo lo que ocurre en el proyecto: decisiones tomadas, riesgos identificados, comunicaciones al equipo, conocimiento técnico adquirido y el estado actual del desarrollo.

Está gestionado por P5 (responsable del proyecto) y se actualiza de forma continua a lo largo de los 30 días de desarrollo.

## Proyecto activo

**HookSuite** — Herramienta de auditoría de seguridad web con Inteligencia Artificial  
**Entrega:** 25 de Mayo de 2026  
**Repositorio de código:** github.com/FeetLovers/Proyecto-Evolve

## Estructura del repositorio

```
manager-proyectos/
├── proyecto/        ← Visión, arquitectura, scopes y plan del proyecto
├── equipo/          ← Perfiles del equipo y dinámicas
├── decisiones/      ← Registro cronológico de todo lo decidido
├── riesgos/         ← Mapa de riesgos activos y mitigados
├── comunicaciones/  ← Documentos e instrucciones generadas para el equipo
├── conocimiento/    ← Investigación técnica e información de referencia
└── conflictos/      ← Registro y resolución de tensiones (vacío hasta ahora)
```

## Cómo usar este repositorio

Cada documento sigue el formato:
```
---
fecha: YYYY-MM-DD
tipo: Decisión / Riesgo / Comunicación / Conflicto / Conocimiento / Avance
proyecto: HookSuite
---
```

Cuando ocurra algo relevante en el proyecto — una decisión, un riesgo nuevo, una comunicación al equipo, un conflicto — se documenta aquí antes de que pase más tiempo.
